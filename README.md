# SANDRA

SANDRA è il runtime deterministico incaricato di mantenere l'equilibrio
dell'Habitat.

## Knowledge canonica

Path assoluto su SANDRA:

`/opt/sandra/knowledge`

La Knowledge conserva sorgenti, decisioni, specifiche, roadmap, stato
progettuale e Journal. Non contiene secret, artefatti o stato operativo
dinamico dell'Habitat.

## Ordine di lettura

1. `PROJECT_CHARTER.md`
2. `ARCHITECTURE.md`
3. `CURRENT_STATE.md`
4. `NEXT_TASK.md`
5. specifiche e ADR richiamate dal tassello corrente
