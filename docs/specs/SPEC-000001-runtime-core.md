# SPEC-000001 — Runtime Core Contract

Il Core espone esclusivamente:

- `sandra_begin`
- `sandra_log`
- `sandra_require_command`
- `sandra_require_file`
- `sandra_assert`
- `sandra_capture`
- `sandra_fail`
- `sandra_end`

Il Core non contiene logica di provider, policy o amministrazione degli
oggetti dell'Habitat.
