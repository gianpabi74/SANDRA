# SPEC-000002 — Knowledge Contract

La Knowledge canonica risiede in:

`/opt/sandra/knowledge`

Deve contenere:

- documentazione umana;
- specifiche;
- ADR;
- sorgenti;
- Journal;
- roadmap;
- stato progettuale;
- prossimo tassello.

Non deve contenere:

- password;
- token;
- chiavi private;
- artefatti;
- log operativi completi;
- dump dello stato runtime.
