# ADR-0016 — Planning Use Case

## Stato

Accepted and immutable.

Candidate generated by R3-000016A, verified against its exact exported

## Decisione candidata

SANDRA uses the existing canonical capability `planning` and the existing
Domain resource kind `ExecutionPlan`.

`BuildExecutionPlan` composes one ordered and bounded plan from a certified
Policy Decision.

Planning owns:

- preconditions;
- ordered actions;
- postconditions;
- recovery;
- execution limits.

Planning excludes:

- action execution;
- technology calls;
- policy evaluation.
