# ADR-0012 — Evidence Qualification Use Case

## Stato

Accepted and immutable.

Candidate generated by R3-000012A, exported and verified from its exact
repository state, then published by R3-000012B.

## Decisione candidata

SANDRA separates:

- authority levels;
- qualification outcomes;
- confidence;
- provenance;
- integrity;
- validity.

`candidate` and `conflicting` are removed from `AuthorityLevel`.

The constitutional authority vocabulary is:

- unknown;
- observational;
- corroborated;
- authoritative.

Conflict remains a qualification outcome and not an authority level.

## Boundaries

Evidence Qualification does not:

- modify authoritative state;
- evaluate execution policy;
- create plans;
- execute actions;
- discard conflicting evidence.
