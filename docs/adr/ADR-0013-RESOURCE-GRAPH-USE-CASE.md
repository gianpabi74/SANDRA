# ADR-0013 — Resource Graph Use Case

## Stato

Accepted and immutable.

Candidate generated by R3-000013A, verified against its exact exported
manifest and published by R3-000013B.

## Decisione candidata

SANDRA uses the existing canonical Domain resources:

- `ManagedObject`;
- `Relationship`;
- `ResourceEnvelope`;
- `ResourceKind`.

The Application Layer does not duplicate or replace these types.

`QueryResourceGraph` retrieves one bounded immutable graph snapshot through
`ResourceGraphReader`.

## Boundaries

The use case owns:

- bounded relationship retrieval;
- dependency traversal;
- peer traversal;
- containment traversal;
- impact-reference retrieval;
- request and result correlation.

The use case does not:

- mutate authoritative state;
- evaluate policy;
- create plans;
- execute actions;
- call technology-specific topology APIs;
- model the entire Habitat as one atomic aggregate.
