# ADR-0009 — Canonical Capability Map V1

## Stato

Accepted and immutable.

Contenuto generato da R3-000009E1, validato indipendentemente da
R3-000009E2R e pubblicato costituzionalmente da R3-000009E3.

## Decisione

SANDRA separa:

- core capability indipendenti dalla tecnologia;
- famiglie operative implementate mediante adapter.

Prodotti, protocolli e trasporti non appartengono al linguaggio del nucleo.

## Conseguenze

- una nuova tecnologia aggiunge un adapter;
- una nuova capability costituzionale richiede una ADR sostitutiva;
- Habitat è rappresentato dal Resource Graph;
- Experience Learning resta subordinata a policy e sicurezza;
- il successo di un comando non sostituisce la verifica.
